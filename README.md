# Job Tracker

A web-based job application tracking platform designed to help students and job seekers stay organized throughout their job search journey.

## Overview

Job Tracker enables users to manage and monitor job applications in one place, making it easier to track progress, stay consistent, and remain motivated during the application process.

## Features

- Track job applications efficiently
- Monitor application progress
- Stay organized throughout the job search
- User-friendly and responsive interface
- Motivation-focused workflow for job seekers
- Centralized application management

## Why I Built This

As a student navigating the competitive job market, I recognized the need for a simple and effective tool to manage multiple job applications. This project was created to help job seekers maintain visibility over their application process and improve their job search productivity.

## Technologies Used

- Frontend: React.js
- Styling: CSS / Tailwind CSS
- Deployment: Netlify
- Version Control: Git & GitHub
